# Nimble Gravity – React Challenge

Mini aplicación desarrollada en React y TypeScript como parte del proceso de selección para el rol de Junior Fullstack Developer.

El objetivo del proyecto es consumir la API provista para:
- obtener los datos del candidato
- listar las posiciones disponibles
- permitir la postulación a una posición enviando la URL del repositorio

El enfoque está puesto en claridad de código, correcto manejo del flujo y tratamiento de errores, priorizando funcionalidad por sobre estilos visuales complejos.

## Tecnologías
- React
- TypeScript
- Vite
- Fetch API

## Funcionalidades
- Obtención de datos del candidato por email
- Listado dinámico de posiciones
- Postulación a una posición mediante API
- Manejo de estados de carga y error
- Confirmación de postulación enviada

## Ejecución del proyecto

```bash
npm install
npm run dev

La aplicación se ejecuta en entorno local y consume directamente la API indicada en el challenge.
